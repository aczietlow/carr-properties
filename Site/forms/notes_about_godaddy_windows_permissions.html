Godaddy-Windows Permissions Instructions

1. log in to godaddy.com

2. select "my hosting account" under "hosting & servers" option on the
main menu

3. click on "manage account" next to the domain name you installed your web forms package

4. click on "your files - manage site content"

5. select the folder "forms" or whichever folder you chose to install bwf

6. click on "permissions"

7. check the boxes:

    Read         (Directory contents are visible to users)
    Write (Applications can write to this directory)

8. click ok

9. repeat steps 5-8 for the directory forms/temp
